Matrizes — Conceitos Fundamentais

Base para o estudo de determinantes

Notação e Terminologia
Uma matriz A m×n é uma tabela retangular com m linhas e n colunas. O elemento da linha i, coluna j é denotado aij.
  • Matriz quadrada: m = n — base para determinantes
  • Matriz identidade I: diagonal principal = 1, demais = 0
  • Matriz transposta AT: linhas e colunas trocadas
  • Matriz nula: todos os elementos iguais a zero
Exemplo — Matriz 3×3
A =
2-13
04-2
15-3
📌 Lembrete: a diagonal principal contém os elementos a11, a22, a33 (em laranja na matriz acima).
⚙️ Operações que você já conhece: soma de matrizes, produto por escalar, produto matricial AB — e suas propriedades (associativa, distributiva). O determinante estende esse arsenal!
Bloco 1 · Determinantes

O que é o Determinante?

Definição intuitiva e formal para matrizes de ordem 1, 2 e 3

O determinante é um escalar associado a toda matriz quadrada A, denotado det(A) ou |A|. Geometricamente, representa a área (2D) ou o volume (3D) do paralelepípedo formado pelos vetores-coluna da matriz — com sinal que indica a orientação.
Ordem 1
det([a]) = a
Ordem 2
|a b|
|c d| = ad − bc
Ordem 3
Regra de Sarrus
(próximo slide)
Exemplo — Ordem 2
A = [3 2]
    [1 4]

det(A) = 3·4 − 2·1
       = 12 − 2 = 10
Interpretação Geométrica
  • det(A) = 0 → vetores-coluna são linearmente dependentes
  • det(A) ≠ 0 → A é invertível (não-singular)
  • |det(A)| = área do paralelogramo (2D)
  • Sinal negativo = orientação invertida

Propriedades Fundamentais

Ferramentas para simplificar o cálculo de qualquer determinante

  • P1 — Identidade: det(I) = 1
  • P2 — Troca de linhas: trocar duas linhas muda o sinal do determinante
  • P3 — Múltiplo escalar: multiplicar uma linha por k multiplica det por k
  • P4 — Linearidade: somar múltiplo de uma linha a outra não altera det
  • P5 — Linha nula: se uma linha (ou coluna) é toda zero → det = 0
  • P6 — Linhas iguais: se duas linhas são iguais → det = 0
  • P7 — Produto: det(AB) = det(A)·det(B)
  • P8 — Transposta: det(AT) = det(A)
  • P9 — Inversa: det(A−1) = 1/det(A)
  • P10 — Triangular: det = produto dos elementos da diagonal principal
  • Atenção: det(A + B) ≠ det(A) + det(B) em geral!
🏆 Propriedade mais usada na prática (P10): em matrizes triangulares superiores ou inferiores, o determinante é simplesmente o produto dos elementos da diagonal. A eliminação gaussiana explora isso!

Regra de Sarrus — Ordem 3

Método mnemônico para calcular det de matrizes 3×3

A Fórmula
det(A) =
+ a₁₁·a₂₂·a₃₃
+ a₁₂·a₂₃·a₃₁
+ a₁₃·a₂₁·a₃₂
− a₁₃·a₂₂·a₃₁
− a₁₁·a₂₃·a₃₂
− a₁₂·a₂₁·a₃₃
💡 Mnemônico: repita as duas primeiras colunas à direita da matriz. As 3 diagonais principais são positivas; as 3 diagonais secundárias são negativas.
Exemplo Resolvido
A = [1 2 3] [4 5 6] [7 8 9] + (1·5·9) = +45 + (2·6·7) = +84 + (3·4·8) = +96 − (3·5·7) = −105 − (1·6·8) = −48 − (2·4·9) = −72 det = 45+84+96−105−48−72 = 0
⚠️ det = 0 → as linhas são linearmente dependentes (perceba que L3 = L1 + L2... não exatamente, mas a matriz é singular neste caso!)
🔖 Atenção: a Regra de Sarrus é válida apenas para matrizes 3×3. Para ordens maiores, usa-se o Teorema de Laplace (desenvolvimento por cofatores) — tema da próxima aula.
Prática

Exemplo Prático — Passo a Passo

Calculando det(A) de uma matriz 3×3 com Sarrus

Dado o problema
A =
23-1
1-24
-312
Calcule det(A) usando a Regra de Sarrus.
Resolução
Diagonais principais (+)
(2)(−2)(2) = −8
(3)(4)(−3) = −36
(−1)(1)(1) = −1
Soma: −45
Diagonais secundárias (−)
(−1)(−2)(−3) = −6
(2)(4)(1) = +8
(3)(1)(2) = +6
Soma: +8 → −8
Resultado Final
det(A) = (−45) − (8) = −53
✅ Verificação: det(A) = −53 ≠ 0, portanto a matriz A é invertível. Isso significa que o sistema linear Ax = b admite solução única para qualquer vetor b.
Conexão Teórica

Determinante e Sistemas Lineares

Por que o determinante importa para a engenharia?

Um sistema Ax = b admite solução única se e somente se det(A) ≠ 0. Esse fato conecta diretamente o determinante à existência e unicidade de soluções — algo central na análise estrutural.

det(A) ≠ 0

Sistema possui solução única. A é invertível. Os vetores-coluna são LI.

det(A) = 0

Sistema sem solução ou com infinitas soluções. A é singular. Vetores-coluna são LD.

Teorema de Cramer

Se det(A) ≠ 0, a solução do sistema é:
xi = det(Ai) / det(A)
🏗️ Engenharia Civil: ao resolver sistemas de equações de equilíbrio de estruturas (MEF, treliças), o determinante da matriz de rigidez indica se a estrutura é estaticamente determinada.
