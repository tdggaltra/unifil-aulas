🎯 Contexto Curricular

Esta disciplina é a continuação direta de Lógica de Programação e Algoritmos I. O aluno já domina variáveis, operadores, E/S e estruturas de controle (if/else, while, for, do-while). Agora avança para organização de dados em coleções e modularização de código — pilares essenciais de qualquer linguagem e paradigma de programação.

🗃️ Unidade I — Vetores

Arranjos unidimensionais: armazenar e processar coleções de dados com índices. Busca e ordenação.

📊 Unidade II — Matrizes

Arranjos bidimensionais: tabelas, grade de dados, operações matriciais e casos especiais.

🔧 Unidade III — Sub-rotinas

Funções e procedimentos: modularização, parâmetros, retorno, escopo e recursividade.

// Exemplo em C — leitura e cálculo de média de vetor #include <stdio.h> int main() { int n = 5; float notas[5], soma = 0; for (int i = 0; i < n; i++) { scanf("%f", &notas[i]); // lê cada elemento soma += notas[i]; } printf("Média: %.2f\n", soma / n); return 0; }

Arranjos Bidimensionais

📘 Aulas 08-09 · Fundamentos

  • → Conceito: tabela de L linhas × C colunas

⚙️ Aulas 10-11 · Operações

  • → Transposição: linhas viram colunas
  • → Multiplicação de matrizes (produto interno)
  • → Matriz identidade
  • → Matriz triangular inferior/superior
  • → Exercícios integrados e revisão
// Percurso e soma de uma matriz 3x3 em C int mat[3][3] = {{1,2,3},{4,5,6},{7,8,9}}; int soma = 0; for (int i = 0; i < 3; i++) { // percorre linhas for (int j = 0; j < 3; j++) { // percorre colunas soma += mat[i][j]; } } // Diagonal principal: mat[i][i] | Diagonal secundária: mat[i][N-1-i]

🗃️ Vetores → Analogia

Um vetor é como uma fila de caixas numeradas — cada uma guarda um valor e você as acessa pelo número da posição.

📊 Matrizes → Analogia

Uma matriz é uma planilha: linha indica a linha da tabela, coluna indica a coluna — você precisa dos dois para achar o dado.

🔧 Sub-rotinas → Analogia

Uma função é como uma receita de bolo: tem ingredientes (parâmetros), segue um processo e entrega um resultado (return).
