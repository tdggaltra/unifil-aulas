🎯 De percurso a operação

Na Aula 08 aprendemos a navegar pela matriz (percurso, soma por linha/coluna, máximo). Hoje vamos operar sobre matrizes: combinar duas matrizes para produzir uma terceira. São as operações mais usadas em computação gráfica, machine learning, processamento de imagens e sistemas de equações lineares.

📋 As 3 operações de hoje

  • Soma A+B: elemento a elemento → C[i][j] = A[i][j] + B[i][j]
  • Transposição Aᵀ: troca linhas por colunas → T[j][i] = A[i][j]
  • Multiplicação A×B: produto escalar linha×coluna → 3 laços aninhados!

⚠️ Pré-requisito de dimensões — fundamental!

Soma: A e B devem ter as mesmas dimensões (L×C).
Transposição: qualquer A[L×C] → T[C×L] (sem restrição).
Multiplicação: nº de colunas de A = nº de linhas de B. A[L×M] × B[M×N] = C[L×N].

🔄 Transposição de Matriz

T[j][i] = A[i][j] — linhas viram colunas, colunas viram linhas

📐 Definição

A transposta de uma matriz A[L×C] é a matriz T[C×L] onde cada linha de A vira uma coluna de T. As dimensões se invertem: A tem L linhas e C colunas → T tem C linhas e L colunas.

int[][] A = {{1,2},{3,4},{5,6}}; int L=A.length, C=A[0].length; int[][] T = new int[C][L]; // invertido! for(int i=0;i<L;i++) for(int j=0;j<C;j++) T[j][i] = A[i][j]; // T.length==C, T[0].length==L for(int[] row : T) { for(int v : row) System.out.printf("%4d",v); System.out.println(); }

💡 Pontos-chave da Transposição

  • A atribuição é T[j][i] = A[i][j] — os índices se invertem!
  • T tem dimensões C×L (inverso de A que é L×C)
  • Para exibir T: laço externo vai até C (linhas de T), interno até L
  • Para matriz quadrada: L==C → T tem o mesmo tamanho
  • Transposta dupla: (Aᵀ)ᵀ = A — volta ao original
A[i][j]→T[j][i]
A[0][0]=1→T[0][0]=1
A[0][1]=2→T[1][0]=2
A[1][0]=3→T[0][1]=3
A[1][1]=4→T[1][1]=4
A[2][0]=5→T[0][2]=5
A[2][1]=6→T[1][2]=6

✖️ Multiplicação de Matrizes — Conceito

Produto escalar de linhas de A com colunas de B

C[i][j] = soma de A[i][k] × B[k][j] para todo k. A[L×M] × B[M×N] = C[L×N]. O nº de colunas de A deve ser igual ao nº de linhas de B!

⚠️ Condição de compatibilidade

  • A[L×M] × B[M×N] = C[L×N]
  • nº colunas de A = nº linhas de B = M
  • Se não compat. → multiplicação indefinida!
  • A[2×3] × B[3×2] = C[2×2] ✅
  • A[2×3] × B[2×3] = indefinido ❌

🔑 Fórmula e 3 laços

  • C[i][j] = Σ A[i][k] × B[k][j], k=0..M-1
  • Laço i: percorre linhas de A (e de C)
  • Laço j: percorre colunas de B (e de C)
  • Laço k: percorre colunas de A = linhas de B
  • Complexidade: O(L×M×N) ≈ O(n³) para quadradas

✖️ Multiplicação de Matrizes — Implementação

Três laços aninhados — o padrão mais importante de matrizes

// A[2×3] × B[3×2] = C[2×2] programa { funcao inicio() { inteiro LA=2, M=3, LB=2 inteiro A[2][3] = {{1,2,3},{4,5,6}} inteiro B[3][2] = {{7,8},{9,10},{11,12}} inteiro C[2][2] // Inicializa C com zeros (obrigatório!) para (inteiro i=0; i<LA; i++) para (inteiro j=0; j<LB; j++) C[i][j] = 0 // Multiplicação: 3 laços aninhados para (inteiro i=0; i<LA; i++) { // linhas de A para (inteiro j=0; j<LB; j++) { // colunas de B para (inteiro k=0; k<M; k++) { // índice comum C[i][j] = C[i][j] + A[i][k] * B[k][j] } } } // Exibir C para (inteiro i=0; i<LA; i++) { para (inteiro j=0; j<LB; j++) escreva(C[i][j] + " ") escreva("\n") } } }
#include <stdio.h> #define LA 2 // linhas de A e C #define M 3 // colunas de A = linhas de B #define LB 2 // colunas de B e C int main() { int A[LA][M] = {{1,2,3},{4,5,6}}; int B[M][LB] = {{7,8},{9,10},{11,12}}; int Cm[LA][LB]; /* Inicializar C com zeros — OBRIGATÓRIO! */ for(int i=0;i<LA;i++) for(int j=0;j<LB;j++) Cm[i][j]=0; /* Multiplicação: i × j × k */ for(int i=0; i<LA; i++) // linhas de A/C for(int j=0; j<LB; j++) // colunas de B/C for(int k=0; k<M; k++) // índice comum Cm[i][j] += A[i][k] * B[k][j]; /* Exibir */ printf("A × B =\n"); for(int i=0;i<LA;i++) { for(int j=0;j<LB;j++) printf("%6d",Cm[i][j]); printf("\n"); } return 0; }
public class MultMatriz { public static void main(String[] args) { int[][] A = {{1,2,3},{4,5,6}}; int[][] B = {{7,8},{9,10},{11,12}}; int LA=A.length, M=B.length, LB=B[0].length; int[][] C = new int[LA][LB]; // já zerado for(int i=0;i<LA;i++) for(int j=0;j<LB;j++) for(int k=0;k<M;k++) C[i][j] += A[i][k] * B[k][j]; for(int[] row : C) { for(int v : row) System.out.printf("%6d",v); System.out.println(); } } }

⚠️ Inicializar C com zeros — SEMPRE!

Em C, variáveis locais contêm lixo de memória. Se Cm não for zerado, o acumulador Cm[i][j] += A[i][k]*B[k][j] somará sobre lixo, produzindo resultados errados. Em Java, new int[LA][LB] já inicializa com zeros — não precisa do laço extra.

💡 O papel do laço k

O laço k é o "índice comum" que percorre as colunas de A e as linhas de B simultaneamente. É ele que faz o produto escalar de cada linha de A com cada coluna de B. Sem o laço k não há multiplicação matricial!

⚖️ Comparativo das 3 Operações

Dimensões, laços, fórmulas e restrições

Operação Entrada Saída Fórmula Laços Restrição
➕ Soma A+B A[L×C], B[L×C] C[L×C] C[i][j] = A[i][j]+B[i][j] 2 aninhados (i,j) A e B mesma dimensão
🔄 Transposta A[L×C] T[C×L] T[j][i] = A[i][j] 2 aninhados (i,j) Nenhuma
✖️ Multiplicação A[L×M], B[M×N] C[L×N] C[i][j] += A[i][k]*B[k][j] 3 aninhados (i,j,k) cols(A) == rows(B)

➕ Quando usar soma

  • Combinar tabelas do mesmo tamanho
  • Acumular resultados de iterações
  • Computação gráfica: blend de imagens
  • Estatística: soma de frequências

🔄 Quando usar transposta

  • Converter entre representações linha/coluna
  • Álgebra linear: resolver sistemas (Gauss)
  • Machine learning: backpropagation
  • Preparar dados para multiplicação

✖️ Quando usar multiplicação

  • Transformações geométricas (rotação, escala)
  • Redes neurais: camadas de pesos
  • Sistemas de equações lineares
  • Grafos: caminhos de comprimento k

🚨 Diferença crítica: soma vs multiplicação

Na soma: C[i][j] = A[i][j] + B[i][j] — acessa o mesmo par [i][j] nas duas matrizes. 2 laços.
Na multiplicação: C[i][j] += A[i][k] * B[k][j] — A usa i+k, B usa k+j. O índice k é o elo! 3 laços. C deve ser zerado antes!

🔥 Ex 03 ★★★ — Multiplicação

Leia A[2][3] e B[3][2]. Calcule C=A×B. Depois calcule D=B×A (3×3). Observe que C ≠ D — a multiplicação matricial não é comutativa!

A×B = C[2×2] (funciona)
B×A = D[3×3] (funciona)
C ≠ D → A×B ≠ B×A ❗
laboratório — programa completo com as 3 operações
#include <stdio.h>
#define L 2
#define C 3

void exibir(int m[][C], int l, int c) {
    for(int i=0;i<l;i++){for(int j=0;j<c;j++) printf("%5d",m[i][j]);printf("\n");}
}

int main() {
    int A[L][C], B[L][C], Soma[L][C], Sub[L][C];

    printf("Digite A (%dx%d): ",L,C);
    for(int i=0;i<L;i++) for(int j=0;j<C;j++) scanf("%d",&A[i][j]);
    printf("Digite B (%dx%d): ",L,C);
    for(int i=0;i<L;i++) for(int j=0;j<C;j++) scanf("%d",&B[i][j]);

    for(int i=0;i<L;i++) for(int j=0;j<C;j++) {
        Soma[i][j] = A[i][j]+B[i][j];
        Sub[i][j]  = A[i][j]-B[i][j];
    }
    printf("A+B:\n"); exibir(Soma,L,C);
    printf("A-B:\n"); exibir(Sub, L,C);
    return 0;
}

🎯 Resumo e Próxima Aula

Três operações dominadas — a base da Álgebra Linear computacional!

Soma, transposição e multiplicação são as operações que constroem todo o mundo de computação gráfica, IA e sistemas lineares! 🔢

✅ Soma A+B — 2 laços, elemento a elemento

C[i][j] = A[i][j] + B[i][j]. Mesmas dimensões obrigatório. Complexidade O(L×C).

✅ Transposição Aᵀ — 2 laços, índices invertidos

T[j][i] = A[i][j]. A[L×C] → T[C×L]. Sem restrição de dimensão. O(L×C).

✅ Multiplicação A×B — 3 laços aninhados

C[i][j] += A[i][k]*B[k][j]. Requer cols(A)=rows(B). C[L×N]. Inicializar C com zeros! O(L×M×N).

✅ Propriedades importantes

Soma é comutativa (A+B=B+A). Multiplicação NÃO é comutativa (A×B ≠ B×A em geral). (Aᵀ)ᵀ = A.

✅ Em C: sempre zere a matriz resultado da multiplicação!

for(i)for(j) C[i][j]=0; ou int C[L][N] = {0}; antes dos 3 laços. Em Java, new int[L][N] já zera.
