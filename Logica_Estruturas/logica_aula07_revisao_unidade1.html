🗺️ Mapa Mental — Unidade I

Tudo que aprendemos sobre vetores em um único slide

📦 VETOR = coleção de elementos do mesmo tipo em memória contígua, indexada a partir de 0

📌 Conceito (Aula 01)

  • Declaração: tipo v[N];
  • Índices: 0 até N-1
  • Percurso: for(i=0;i<N;i++)
  • scanf: scanf("%d",&v[i])
  • Java: tipo[] v = new tipo[N]

📊 Estatísticas (Aula 02)

  • Soma: acumulador = 0
  • Média: (float)soma / N
  • Máx/Mín: inicia com v[0]
  • Laço máx/mín: começa em i=1
  • Contagem: cont++ na condição

🧪 Laboratório (Aula 03)

  • Ex 01: par/ímpar com %
  • Ex 02: soma e média de notas
  • Ex 03: máx, mín, amplitude
  • Ex 04: aprovados/recuperação
  • Ex 05-06: acima da média, estação

🔍 Busca (Aula 04)

  • Linear: percorre i=0..N-1 · O(n)
  • Retorna posição ou -1
  • Binária: divide ao meio · O(log n)
  • Exige vetor ordenado!
  • Variáveis: esq · dir · meio

🫧 Ordenação (Aula 05)

  • Bubble: compara adjacentes, troca · O(n²)
  • Otimização: flag trocou
  • Selection: seleciona mínimo · O(n²)
  • Max N-1 trocas totais
  • Swap sempre usa temp

🔤 Strings (Aula 06)

  • char s[N] + terminador \0
  • Tamanho mínimo = strlen+1
  • strlen · strcpy · strcmp · strcat
  • Nunca use == para comparar!
  • fgets remove \n do final

📊 Tabela Comparativa — 5 Algoritmos

Quando usar cada um — referência rápida para a prova

Algoritmo Complexidade Pré-requisito Inicialização-chave Quando usar
Soma / Média O(n) Nenhum soma = 0 · cast (float) Calcular totais e médias de coleção
Máximo / Mínimo O(n) Nenhum maior = v[0] · laço desde i=1 Encontrar extremos — nunca inicie com 0!
Busca Linear O(n) Nenhum Retorna posição ou -1 Vetor pequeno ou desordenado
Busca Binária O(log n) Vetor ordenado! esq=0, dir=N-1, meio=(esq+dir)/2 Vetor grande e estático já ordenado
Bubble Sort O(n²) · best O(n) Nenhum Flag trocou=0 por passagem Pequenos ou quase ordenados
Selection Sort O(n²) Nenhum posMin=i a cada passagem Quando trocas são custosas (≤ N-1 trocas)

🎯 Regra de escolha rápida

Busca em vetor ordenado grande → binária.
Busca em vetor pequeno ou desordenado → linear.
Ordenar poucas vezes e buscar muito → ordena (O(n²)) uma vez, busca binária (O(log n)) sempre.
Para N<50 qualquer algoritmo funciona; para N>10.000 use qsort()/Arrays.sort().

🔗 Conexões entre algoritmos

  • Máximo da Aula 02 é usado no Selection Sort (Aula 05)
  • Busca binária (Aula 04) exige a ordenação (Aula 05)
  • Ranking com Selection Sort pode usar strcmp (Aulas 05+06)
  • Acima da média (Aula 03) usa 2 passagens — como busca + contagem
  • strlen percorre igual ao algoritmo de busca linear!

💻 Exercício Integrado 01 — Sistema de Notas

Combina: leitura · soma/média · máx · contagem · ordenação

Enunciado: leia N notas de alunos com nome e nota. Calcule média da turma. Exiba quem ficou acima da média. Ordene por nota (decrescente) e mostre o ranking. Realize busca por nome.
sistema_notas.c — solução integrada completa
#include <stdio.h>
#include <string.h>
#define N 5

/* 1. Ordenação decrescente por nota (Selection Sort) */
void ordenar(char nomes[][50], float notas[], int n) {
    for(int i=0;i<n-1;i++) {
        int pm=i;
        for(int j=i+1;j<n;j++) if(notas[j]>notas[pm]) pm=j;
        if(pm!=i) {
            float t=notas[i]; notas[i]=notas[pm]; notas[pm]=t;
            char tmp[50]; strcpy(tmp,nomes[i]); strcpy(nomes[i],nomes[pm]); strcpy(nomes[pm],tmp);
        }
    }
}

/* 2. Busca linear por nome */
int buscarNome(char nomes[][50], int n, char chave[]) {
    for(int i=0;i<n;i++) if(strcmp(nomes[i],chave)==0) return i;
    return -1;
}

int main() {
    char  nomes[N][50];
    float notas[N], soma=0, media;

    /* Leitura */
    for(int i=0;i<N;i++) {
        printf("Nome %d: ",i+1); scanf("%s",nomes[i]);
        printf("Nota %d: ",i+1); scanf("%f",¬as[i]);
        soma+=notas[i];
    }
    media=(float)soma/N;
    printf("\nMédia: %.2f\n",media);

    /* Acima da média */
    printf("Acima da média: ");
    for(int i=0;i<N;i++) if(notas[i]>media) printf("%s(%.1f) ",nomes[i],notas[i]);

    /* Ordenar e exibir ranking */
    ordenar(nomes,notas,N);
    printf("\n\nRanking:\n");
    for(int i=0;i<N;i++) printf("%dº %-15s %.1f\n",i+1,nomes[i],notas[i]);

    /* Busca por nome */
    char busca[50]; printf("\nBuscar aluno: "); scanf("%s",busca);
    int pos=buscarNome(nomes,N,busca);
    if(pos!=-1) printf("Ranking %dº: nota %.1f\n",pos+1,notas[pos]);
    else         puts("Não encontrado.");
    return 0;
}

💻 Exercício Integrado 03 — Processamento de Strings

Combina: strings · percurso · ordenação · contagem

📋 Enunciado — Dicionário de Palavras

Leia 8 palavras do usuário (char palavras[8][50]). Para cada uma, calcule seu comprimento. Exiba a maior palavra (em tamanho). Ordene as palavras em ordem alfabética (Selection Sort + strcmp). Verifique quais são palíndromos. Por fim, permita busca binária no vetor ordenado usando strcmp.

dicionario.c — função palíndromo
#include <stdio.h>
#include <string.h>
#include <ctype.h>   // tolower()
#define N 8

/* Verifica palíndromo ignorando maiúsculas */
int ehPalindromo(char s[]) {
  int esq=0, dir=strlen(s)-1;
  while(esq < dir) {
    if(tolower(s[esq]) != tolower(s[dir])) return 0;
    esq++; dir--;
  }
  return 1;
}

/* Busca binária em vetor de strings ordenado */
int buscaBin(char v[][50], int n, char chave[]) {
  int esq=0, dir=n-1;
  while(esq<=dir) {
    int m=(esq+dir)/2;
    int cmp=strcmp(v[m],chave);
    if(!cmp) return m;
    if(cmp<0) esq=m+1; else dir=m-1;
  }
  return -1;
}

int main() {
  char palavras[N][50];
  int  maiorTam=0, iMaior=0;

  for(int i=0;i<N;i++) {
    printf("Palavra %d: ",i+1);
    scanf("%s",palavras[i]);
    int t=strlen(palavras[i]);
    if(t>maiorTam){ maiorTam=t; iMaior=i; }
    if(ehPalindromo(palavras[i]))
      printf("  → palíndromo!\n");
  }
  printf("\nMaior: %s (%d chars)\n",palavras[iMaior],maiorTam);
  /* TODO: Selection Sort + busca binária */
  return 0;
}

✅ Conceitos integrados

  • strlen → encontrar maior palavra
  • strcmp → ordenar e buscar
  • Selection Sort → ordenar alfabeticamente
  • Busca binária → após ordenar
  • Algoritmo palíndromo → dois índices
  • tolower() → ignorar maiúsculas

🔗 strcmp na busca binária

  • strcmp(v[meio],chave) == 0 → encontrou
  • strcmp(v[meio],chave) < 0 → meio vem antes → esq = meio+1
  • strcmp(v[meio],chave) > 0 → meio vem depois → dir = meio-1
  • Funciona igual à busca binária de inteiros, só muda o comparador!
