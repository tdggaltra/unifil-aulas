Aula 20 — Structs e Registros

🗂️

Agrupe Dados Diferentes em Um Tipo Só!

Uma struct (registro) permite criar tipos compostos — reunir campos de tipos diferentes (nome, idade, CPF, salário) em uma única variável com significado real no domínio do problema. É o primeiro passo da modelagem de dados!

🏗️ Declaração e Uso Básico

Como criar um novo tipo, declarar variáveis e acessar campos

Struct = ficha cadastral no código. Cada campo é um dado; a struct é o formulário completo! 📋

🔵 C — declarar e usar struct

// Definição do tipo (fora de main) struct Aluno { char nome[50]; int matricula; float media; char turma; }; // Declarar variáveis struct Aluno a1; ← uma variável struct Aluno turmaA[30]; ← vetor de structs // Atribuir campos — operador ponto (.) strcpy(a1.nome, "Maria Silva"); a1.matricula = 20230042; a1.media = 8.75; a1.turma = 'A'; // Ler campo printf("Aluno: %s\n", a1.nome); printf("Média: %.2f\n", a1.media); // Inicialização com literal struct Aluno a2 = {"João",20230043,7.5,'B'};

🔵 C — typedef simplifica o uso

// SEM typedef — verboso struct Aluno a1; struct Aluno turma[30]; // COM typedef — limpo typedef struct { char nome[50]; int matricula; float media; char turma; } Aluno; // Agora usa sem "struct": Aluno a1; ← muito mais limpo! Aluno turma[30]; // Atribuição direta entre structs Aluno a3 = a1; ← copia todos os campos! // (diferente de arrays — structs podem ser copiadas com =)

🆚 Struct vs. variáveis separadas

Sem struct você teria nome[100][50], matricula[100], media[100] — três vetores que se perdem. Com struct: Aluno turma[100] — tudo junto, organizado e passável como parâmetro!

⚙️ Structs em Funções

Passar struct por valor (cópia) ou por ponteiro (referência) — e retornar structs

🔵 C — passar por valor (cópia)

// Recebe cópia — não altera original void exibirAluno(Aluno a) { printf("%-20s %8d %5.2f %c\n", a.nome, a.matricula, a.media, a.turma); } // Retornar struct (nova struct criada) Aluno criarAluno(char *nome, int mat, float med, char t) { Aluno novo; strcpy(novo.nome, nome); novo.matricula = mat; novo.media = med; novo.turma = t; return novo; ← retorna struct! } Aluno a = criarAluno("Ana",2023001,9.0,'A');

🔵 C — passar por ponteiro (modifica original)

// Ponteiro para struct — operador -> void aplicarBonus(Aluno *a, float bonus) { a->media += bonus; ← modifica original! if(a->media > 10.0) a->media = 10.0; } // a->campo equivale a (*a).campo void lerAluno(Aluno *a) { printf("Nome: "); fgets(a->nome, 50, stdin); printf("Matrícula: "); scanf("%d", &a->matricula); printf("Média: "); scanf("%f", &a->media); } // Uso: Aluno a1; lerAluno(&a1); ← passa endereço aplicarBonus(&a1, 0.5);

✅ . vs. ->

a.campo — variável direta
ptr->campo — via ponteiro (equivale a (*ptr).campo)
Regra prática: tem * na declaração? Use ->. Caso contrário, use .

📦 Vetor de Structs — Cadastro Completo

A combinação mais usada na prática: vários registros do mesmo tipo

Vetor de structs = banco de dados na memória! Cada posição é um registro completo. 🗃️

🔵 C — turma de alunos

#define MAX 30 Aluno turma[MAX]; int n = 0; ← qtd. cadastrada // CADASTRAR void cadastrar(Aluno t[], int *n) { if(*n == MAX) { printf("Cheio!\n"); return; } printf("Nome: "); fgets(t[*n].nome,50,stdin); t[*n].nome[strcspn(t[*n].nome,"\n")]=0; printf("Mat: "); scanf("%d",&t[*n].matricula); printf("Média: "); scanf("%f",&t[*n].media); printf("Turma: "); scanf(" %c",&t[*n].turma); (*n)++; printf("✅ Cadastrado!\n"); } // LISTAR void listar(Aluno t[], int n) { printf("%-20s %10s %6s %5s\n", "Nome","Matrícula","Média","T"); printf("%-20s %10s %6s %5s\n", "────────────────────","─────────","─────","─"); for(int i=0;i<n;i++) printf("%-20s %10d %6.2f %5c\n", t[i].nome,t[i].matricula, t[i].media,t[i].turma); }

🔵 C — buscar e calcular média da turma

// BUSCAR por nome int buscarNome(Aluno t[], int n, char *busca) { for(int i=0; i<n; i++) if(strstr(t[i].nome, busca)) return i; ← índice encontrado return -1; ← não encontrado } // ESTATÍSTICAS void estatisticas(Aluno t[], int n) { float soma=0, maior=t[0].media; int aprov=0; Aluno destaque = t[0]; for(int i=0;i<n;i++){ soma += t[i].media; if(t[i].media > maior){ maior = t[i].media; destaque = t[i]; ← copia struct! } if(t[i].media >= 6.0) aprov++; } printf("Média turma : %.2f\n", soma/n); printf("Aprovados : %d/%d\n", aprov, n); printf("Destaque : %s (%.2f)\n", destaque.nome, destaque.media); }

🪆 Structs Aninhadas

Uma struct pode conter outra struct como campo — modelagem mais rica

typedef struct — Data
intdia→ 7
intmes→ 3
intano→ 2026
typedef struct — Funcionario
charnome[60]→ "Ana Lima"
intcpf→ 12345678
floatsalario→ 4500.00
Dataadmissao→ {1, 3, 2020}
Datanascimento→ {15, 6, 1995}

🔵 C — código com struct aninhada

typedef struct { int dia, mes, ano; } Data; typedef struct { char nome[60]; long cpf; float salario; Data admissao; ← struct dentro de struct! Data nascimento; } Funcionario; Funcionario f; strcpy(f.nome, "Ana Lima"); f.salario = 4500.0; // Acessar campo de struct aninhada: f.admissao.dia = 1; ← dois pontos! f.admissao.mes = 3; f.admissao.ano = 2020; printf("%s — admitido em %02d/%02d/%d\n", f.nome, f.admissao.dia, f.admissao.mes, f.admissao.ano);

✅ Encadeamento de pontos

Para campos aninhados, encadeie pontos: f.admissao.dia. Via ponteiro: fp->admissao.dia — o -> é só para o nível do ponteiro; o campo interno usa . normalmente.

💻 Exemplo 1 — Sistema de Estoque com Structs

Modelando produtos reais: cadastro, busca e relatório financeiro

📌 Problema Intermediário

Sistema para uma loja com struct Produto: código, nome, preço, quantidade. Funções: cadastrar, listar, buscar por código, calcular valor total do estoque.

🔵 C — struct e funções

typedef struct { int codigo; char nome[40]; float preco; int qtd; } Produto; #define MAX 50 Produto estoque[MAX]; int total = 0; // Cadastrar produto void cadastrar() { if(total==MAX){printf("Cheio!\n");return;} Produto *p = &estoque[total]; printf("Código: "); scanf("%d",&p->codigo); printf("Nome : "); scanf(" %39[^\n]", p->nome); printf("Preço : "); scanf("%f",&p->preco); printf("Qtd : "); scanf("%d",&p->qtd); total++; printf("✅ Cadastrado!\n"); } // Buscar por código Produto* buscar(int cod) { for(int i=0;i<total;i++) if(estoque[i].codigo==cod) return &estoque[i]; ← retorna ponteiro! return NULL; }

🔵 C — listar e valor total

void listar() { printf("\n%4s %-20s %9s %6s %12s\n", "Cód","Nome","Preço","Qtd","Total"); printf("%s\n","────────────────────────────────────────────"); for(int i=0;i<total;i++) { Produto p = estoque[i]; printf("%4d %-20s R$%7.2f %6d R$%10.2f\n", p.codigo,p.nome,p.preco, p.qtd,p.preco*p.qtd); } } float valorTotal() { float vt = 0; for(int i=0;i<total;i++) vt += estoque[i].preco * estoque[i].qtd; return vt; } // Uso da busca com ponteiro: Produto *p = buscar(101); if(p) { printf("Encontrado: %s\n", p->nome); p->qtd += 10; ← atualiza diretamente! } else printf("Código não encontrado.\n");

💻 Exemplo 2 — Ordenar Vetor de Structs

Bubble Sort adaptado para structs — comparar campo e trocar registros inteiros

📌 Problema Intermediário

Ordenar o vetor de alunos por média (decrescente) e exibir ranking. Usar a função qsort da stdlib e também implementar Bubble Sort manualmente.

🔵 C — Bubble Sort em structs

// Ordenar por média DECRESCENTE void ordenarPorMedia(Aluno t[], int n) { int trocou; for(int i=0; i<n-1; i++) { trocou = 0; for(int j=0; j<n-1-i; j++) { if(t[j].media < t[j+1].media) { // Troca STRUCT inteira: Aluno tmp = t[j]; ← cópia! t[j] = t[j+1]; t[j+1] = tmp; trocou = 1; } } if(!trocou) break; ← otimizado } } // Exibir ranking void ranking(Aluno t[], int n) { ordenarPorMedia(t, n); printf("\n🏆 RANKING DA TURMA\n"); for(int i=0; i<n; i++) printf("%2dº %-20s %.2f %s\n", i+1, t[i].nome, t[i].media, t[i].media>=6?"✅":"❌"); }

🔵 C — qsort com função de comparação

#include <stdlib.h> // Função de comparação para qsort int cmpMedia(const void *a, const void *b) { Aluno *x = (Aluno*)a; Aluno *y = (Aluno*)b; // Decrescente: y - x if(y->media > x->media) return 1; if(y->media < x->media) return -1; return 0; } // Ordenar com qsort: qsort(turma, n, sizeof(Aluno), cmpMedia); // Para ordenar por nome: int cmpNome(const void *a, const void *b){ return strcmp((Aluno*)a->nome, (Aluno*)b->nome); }

🔑 Troca de struct inteira

Diferente de um vetor de int, ao ordenar structs a troca move o registro inteiro — todos os campos juntos. Basta usar uma variável temporária do mesmo tipo: Aluno tmp = t[j]. Isso funciona porque structs suportam cópia com =.

🔢 Enum — Conjuntos de Constantes Nomeadas

Enumerações tornam o código mais legível e seguro — adeus aos "números mágicos"!

enum substitui constantes numéricas por nomes significativos — código que se explica! 📖

🔵 C — enum básico

// Sem enum — confuso int situacao = 2; ← O que é 2? if(situacao == 3) ... ← aprovado? // Com enum — claro! typedef enum { REPROVADO = 0, RECUPERACAO, ← 1 APROVADO, ← 2 APROVADO_DIST ← 3 } Situacao; Situacao sit = APROVADO; if(sit == APROVADO) printf("Parabéns!\n"); // Enum em switch: switch(sit) { case REPROVADO: printf("Rep.\n"); break; case RECUPERACAO: printf("Rec.\n"); break; case APROVADO: printf("Apr.\n"); break; }

🔵 C — enum dentro de struct

typedef enum { MASCULINO, FEMININO, OUTRO } Genero; typedef enum { ATIVO, INATIVO, LICENCA } Status; typedef struct { char nome[60]; float salario; Genero genero; ← enum como campo! Status status; } Funcionario; Funcionario f; strcpy(f.nome, "Carlos"); f.genero = MASCULINO; f.status = ATIVO; // Muito mais legível que: // f.genero = 0; f.status = 0; if(f.status == ATIVO) printf("%s está ativo.\n", f.nome);

✅ Quando usar enum

Use enum sempre que um campo tiver um conjunto fixo e limitado de valores possíveis: dias da semana, meses, estados de um processo, categorias, níveis de acesso. Elimina os "números mágicos" e ativa alertas do compilador para casos não tratados.

💻 Exemplo 3 — Agenda de Contatos Completa

Structs aninhadas, enum, vetor e menu — programa completo e profissional

📌 Problema Avançado

Agenda com struct Contato contendo: nome, telefone, email, Data de aniversário e tipo (AMIGO, TRABALHO, FAMILIA). Menu: adicionar, buscar, listar por tipo, aniversariantes do mês.

🔵 C — tipos e estrutura

typedef enum { AMIGO, TRABALHO, FAMILIA } TipoContato; typedef struct { int dia, mes, ano; } Data; typedef struct { char nome[60]; char telefone[20]; char email[60]; Data aniversario; TipoContato tipo; } Contato; char* tipoStr(TipoContato t) { switch(t){ case AMIGO: return "👫 Amigo"; case TRABALHO: return "💼 Trabalho"; case FAMILIA: return "🏠 Família"; default: return "?"; } }

🔵 C — funções de consulta

// Listar por tipo void listarPorTipo(Contato a[], int n, TipoContato tipo) { printf("\n=== %s ===\n", tipoStr(tipo)); int found = 0; for(int i=0; i<n; i++) if(a[i].tipo == tipo) { printf(" %s — %s\n", a[i].nome, a[i].telefone); found++; } if(!found) printf(" (nenhum)\n"); } // Aniversariantes do mês void aniversariantesMes(Contato a[], int n, int mes) { printf("\n🎂 Aniversários em %02d:\n", mes); for(int i=0; i<n; i++) if(a[i].aniversario.mes == mes) printf(" %s — dia %02d\n", a[i].nome, a[i].aniversario.dia); }

🚨 Erros Comuns com Structs

Armadilhas frequentes — aprenda antes de cair!

❌ Erro 1: Atribuir string com = dentro de struct

Aluno a; // ERRADO: = não copia string! a.nome = "Maria"; ← ERRO de compilação! // CORRETO: strcpy para campos char[] strcpy(a.nome, "Maria"); ← ✅ // EXCEÇÃO: inicialização na declaração Aluno a2 = {"Maria", 1001, 8.5, 'A'}; ← ✅

❌ Erro 2: Usar . com ponteiro de struct

Aluno *p = &turma[0]; // ERRADO: . com ponteiro printf("%s", p.nome); ← ERRO! // CORRETO: -> com ponteiro printf("%s", p->nome); ← ✅ // Ou desrreferenciar antes: printf("%s", (*p).nome); ← ✅ (mas feio)

❌ Erro 3: Vetor de structs sem controle de quantidade

// ERRADO: percorrer além do cadastrado Aluno turma[30]; ← capacidade 30 int n = 5; ← 5 cadastrados for(int i=0; i<30; i++) ← ERRADO! printf("%s\n", turma[i].nome); ← lixo! // CORRETO: usar n (quantidade real) for(int i=0; i<n; i++) ← ✅ printf("%s\n", turma[i].nome);

❌ Erro 4: Comparar structs com ==

Aluno a1, a2; // ERRADO: não funciona em C! if(a1 == a2) ... ← ERRO! // CORRETO: comparar campo por campo if(a1.matricula == a2.matricula && strcmp(a1.nome, a2.nome) == 0) printf("Iguais!\n");

✅ Checklist para structs

☑ Campos char[] usam strcpy?  |  ☑ Ponteiro de struct usa ->?  |  ☑ Vetor percorre até n (cadastrado), não o tamanho total?  |  ☑ Comparação de structs é campo a campo?

🎯 Resumo e Próximos Passos

Structs — o primeiro passo para modelar o mundo real no código!

struct agrupa campos de tipos diferentes em um único tipo com significado real! 🗂️

✅ Conceitos aprendidos

  • struct reúne campos de tipos diferentes
  • typedef struct cria alias limpo para o tipo
  • Acesso com . (variável) e -> (ponteiro)
  • Structs podem ser copiadas com =
  • enum define constantes nomeadas para conjuntos fixos

✅ Exemplos resolvidos

  • Cadastro de alunos — vetor de structs com busca e estatísticas
  • Sistema de estoque — ponteiro de retorno para atualização direta
  • Ordenação — Bubble Sort e qsort em vetor de structs
  • Agenda com structs aninhadas e enum de tipo de contato

⚠️ Erros a evitar

  • Atribuir string com = → use strcpy()
  • Usar . com ponteiro → use ->
  • Percorrer além do quantidade real (n)
  • Comparar structs com == → compare campo a campo
