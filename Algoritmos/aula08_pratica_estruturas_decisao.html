✏️ Exercício 1 — Par ou Ímpar

Estrutura if/else simples com operador módulo

📌 Enunciado

Leia um número inteiro digitado pelo usuário e informe se ele é PAR ou ÍMPAR. Lembre-se: um número é par quando o resto da divisão por 2 é igual a zero.

🔍 Trace Manual — antes de programar!

n = 8:  8 % 2 = 0 → PAR   |   n = 7:  7 % 2 = 1 → ÍMPAR   |   n = 0:  0 % 2 = 0 → PAR

🟢 Portugol

algoritmo "ParImpar" var n: inteiro inicio escreva("Digite um número: ") leia(n) se (n % 2 == 0) entao escreva(n, " é PAR") senao escreva(n, " é ÍMPAR") fimse fimalgoritmo

🔵 C

#include <stdio.h> int main() { int n; printf("Digite um número: "); scanf("%d", &n); if (n % 2 == 0) { printf("%d é PAR\n", n); } else { printf("%d é ÍMPAR\n", n); } return 0; }

🟠 Java

import java.util.Scanner; public class ParImpar { public static void main (String[] args) { Scanner sc = new Scanner(System.in); System.out.print( "Digite um número: "); int n = sc.nextInt(); if (n % 2 == 0) { System.out.println( n + " é PAR"); } else { System.out.println( n + " é ÍMPAR"); } } }

✅ Conceito-chave: Operador Módulo (%)

O operador % retorna o resto da divisão inteira. Todo número par tem resto 0 quando dividido por 2. Números negativos também funcionam: -4 % 2 = 0 → PAR.

✏️ Exercício 2 — Classificação de Triângulo

Múltiplas condições com operadores lógicos

📌 Enunciado

Leia os três lados de um triângulo (a, b, c). Primeiro verifique se é um triângulo válido — a soma de dois lados deve ser sempre maior que o terceiro. Se válido, classifique como: Equilátero (3 lados iguais), Isósceles (2 lados iguais) ou Escaleno (todos diferentes).

🟢 Portugol

algoritmo "Triangulo" var a, b, c: real inicio escreva("Lado A: ") leia(a) escreva("Lado B: ") leia(b) escreva("Lado C: ") leia(c) se ((a+b > c) e (a+c > b) e (b+c > a)) entao se (a == b) e (b == c) entao escreva("Equilátero") senao se (a==b) ou (b==c) ou (a==c) entao escreva("Isósceles") senao escreva("Escaleno") fimse fimse senao escreva("NÃO é triângulo!") fimse fimalgoritmo

🔵 C / 🟠 Java (mesma lógica)

// Versão em C #include <stdio.h> int main() { float a, b, c; scanf("%f %f %f", &a, &b, &c); if((a+b>c) && (a+c>b) && (b+c>a)){ if(a==b && b==c) printf("Equilátero\n"); else if(a==b || b==c || a==c) printf("Isósceles\n"); else printf("Escaleno\n"); } else { printf("NÃO é triângulo!\n"); } return 0; }
LadosVálido?Tipo
3, 3, 3✅Equilátero
5, 5, 3✅Isósceles
3, 4, 5✅Escaleno
1, 2, 10❌Não é triângulo

✏️ Exercício 3 — Validação de Login

Operadores lógicos combinados com if/else

📌 Enunciado

Simule o login de um sistema. O usuário deve digitar usuário e senha. O acesso só é liberado se o usuário for "admin" E a senha for "1234". Exiba mensagem de boas-vindas ou de acesso negado.

🟢 Portugol

algoritmo "Login" var usuario, senha: caractere inicio escreva("Usuário: ") leia(usuario) escreva("Senha: ") leia(senha) se (usuario == "admin") e (senha == "1234") entao escreva("Bem-vindo, Admin!") escreva("Acesso liberado ✔") senao escreva("Acesso negado ✖") escreva("Usuário/senha errado") fimse fimalgoritmo

🔵 C

#include <stdio.h> #include <string.h> int main() { char usuario[50], senha[50]; printf("Usuário: "); scanf("%s", usuario); printf("Senha: "); scanf("%s", senha); if (strcmp(usuario,"admin")==0 && strcmp(senha, "1234") ==0) { printf("Bem-vindo, Admin!\n"); printf("Acesso liberado\n"); } else { printf("Acesso negado!\n"); } return 0; }

🟠 Java

import java.util.Scanner; public class Login { public static void main (String[] args) { Scanner sc = new Scanner(System.in); System.out.print( "Usuário: "); String u = sc.next(); System.out.print( "Senha: "); String s = sc.next(); if (u.equals("admin") && s.equals("1234")) { System.out.println( "Bem-vindo, Admin!"); } else { System.out.println( "Acesso negado!"); } } }

⚠️ Atenção: Comparar Strings

  • → C: use strcmp(a, b) == 0 — o operador == compara endereços de memória, não conteúdo!
  • → Java: use a.equals(b) — nunca use == para comparar objetos String!
  • → Portugol: == funciona normalmente para caractere.

✏️ Exercício 4 — Calculadora de IMC

else if encadeado com faixas de valores reais

📌 Enunciado

Leia o peso (kg) e a altura (m) de uma pessoa. Calcule o IMC (peso / altura²) e classifique segundo a tabela da OMS.

IMCClassificação
Abaixo de 18,5Abaixo do peso
18,5 a 24,9Peso normal ✅
25,0 a 29,9Sobrepeso
30,0 a 39,9Obesidade
40,0 ou maisObesidade grave

🔍 Exemplo de Trace

Peso=70, Alt=1.75:
IMC = 70 / (1.75²) = 70 / 3.0625 ≈ 22.9 → Peso normal ✅

Peso=90, Alt=1.70:
IMC = 90 / (1.70²) = 90 / 2.89 ≈ 31.1 → Obesidade

🟢 Portugol

algoritmo "IMC" var peso, altura, imc: real inicio escreva("Peso (kg): ") leia(peso) escreva("Altura (m): ") leia(altura) imc <- peso / (altura * altura) escreva("IMC: ", imc:5:2) se (imc < 18.5) entao escreva("Abaixo do peso") senao se (imc < 25.0) entao escreva("Peso normal ✅") senao se (imc < 30.0) entao escreva("Sobrepeso") senao se (imc < 40.0) entao escreva("Obesidade") senao escreva("Obesidade grave") fimse fimse fimse fimse fimalgoritmo

✏️ Exercício 4 — IMC em C e Java

Mesmo algoritmo, sintaxe diferente — else if encadeado

🔵 C

#include <stdio.h> int main() { float peso, altura, imc; printf("Peso (kg): "); scanf("%f", &peso); printf("Altura (m): "); scanf("%f", &altura); imc = peso / (altura * altura); printf("IMC: %.2f\n", imc); if (imc < 18.5f) printf("Abaixo do peso\n"); else if (imc < 25.0f) printf("Peso normal\n"); else if (imc < 30.0f) printf("Sobrepeso\n"); else if (imc < 40.0f) printf("Obesidade\n"); else printf("Obesidade grave\n"); return 0; }

🟠 Java

import java.util.Scanner; public class IMC { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Peso (kg): "); double peso = sc.nextDouble(); System.out.print("Altura (m): "); double alt = sc.nextDouble(); double imc = peso / (alt * alt); System.out.printf("IMC: %.2f%n", imc); if (imc < 18.5) System.out.println("Abaixo do peso"); else if (imc < 25.0) System.out.println("Peso normal ✅"); else if (imc < 30.0) System.out.println("Sobrepeso"); else if (imc < 40.0) System.out.println("Obesidade"); else System.out.println("Obesidade grave"); } }

💡 Por que usar else if e não ifs separados?

Com ifs separados, todos seriam testados mesmo depois de encontrar o correto — desperdício e risco de múltiplas saídas. Com else if, ao encontrar a primeira condição verdadeira o programa pula o restante, economizando processamento e garantindo exatamente um resultado.

🏗️ Mini-Projeto — Sistema de Matrícula

Integrando if/else, else if e switch em um programa completo

📌 Especificação do Projeto

Crie um sistema de matrícula universitária que:

  • Leia nome, idade e nota do vestibular (0–100) do candidato
  • Valide se a idade é ≥ 17 anos e a nota é ≥ 60
  • Se aprovado, leia a opção de curso (1-Eng. Software, 2-Sistemas, 3-Ciência da Comp.)
  • Exiba uma mensagem de confirmação de matrícula completa

📐 Estrutura do Algoritmo

  • Passo 1: Ler nome, idade, nota
  • Passo 2: Validar com if (idade ≥ 17 E nota ≥ 60)
  • Passo 3: Dentro do bloco aprovado, ler opção de curso
  • Passo 4: Usar switch para exibir o curso escolhido
  • Passo 5: No bloco reprovado, indicar o motivo

🧪 Casos de Teste

  • João, 20, 75, curso 1 → Matrícula em Eng. Software ✅
  • Ana, 16, 80, — → Reprovado (menor de idade) ❌
  • Pedro, 19, 50, — → Reprovado (nota insuficiente) ❌
  • Maria, 17, 60, curso 3 → Matrícula em Ciência da Comp. ✅

🔑 Decisões combinadas no projeto

if/else → validação de acesso (idade E nota)  |  else if → mensagem de reprovação personalizada por motivo  |  switch → seleção do curso após aprovação

💻 Mini-Projeto — Código em Portugol

Sistema de Matrícula completo passo a passo

algoritmo "Matricula" var nome : caractere idade : inteiro nota : real curso : inteiro inicio escreva("=== SISTEMA DE MATRÍCULA ===") escreva("Nome: ") leia(nome) escreva("Idade: ") leia(idade) escreva("Nota vest.: ") leia(nota) se (idade >= 17) e (nota >= 60) entao escreva("✔ Candidato aprovado!") escreva("1-Eng.Software 2-Sistemas") escreva("3-Ciência Comp. : ") leia(curso) // ... switch abaixo senao escreva("✖ Matrícula negada.") se (idade < 17) e (nota < 60) entao escreva("Motivo: idade e nota.") senao se (idade < 17) entao escreva("Motivo: menor de idade.") senao escreva("Motivo: nota insuficiente.") fimse fimse fimse fimalgoritmo

🟢 Portugol — Parte 2: Seleção de Curso

// Dentro do bloco "aprovado", após ler curso: escolha (curso) caso 1 escreva("Curso: Eng. de Software") escreva("Período: Noturno") escreva("Duração: 4 anos") caso 2 escreva("Curso: Sistemas de Info.") escreva("Período: Noturno") escreva("Duração: 4 anos") caso 3 escreva("Curso: Ciência da Comp.") escreva("Período: Integral") escreva("Duração: 4 anos") outrocaso escreva("Opção inválida!") fimescolha escreva("Matrícula de ", nome, " realizada com sucesso!")

✅ Estruturas usadas no projeto

  • if + && — validação dupla (idade E nota)
  • if/else if/else — motivo de reprovação
  • switch/case — seleção de curso

💻 Mini-Projeto — Código em C e Java

Mesma lógica, sintaxe de produção

🔵 C

#include <stdio.h> int main() { char nome[50]; int idade, curso; float nota; printf("Nome: "); scanf("%s", nome); printf("Idade: "); scanf("%d", &idade); printf("Nota: "); scanf("%f", &nota); if (idade >= 17 && nota >= 60) { printf("Aprovado! Escolha o curso:\n"); printf("1-Eng.SW 2-Sistemas 3-CC: "); scanf("%d", &curso); switch(curso) { case 1: printf("Eng. Software\n"); break; case 2: printf("Sistemas Info.\n"); break; case 3: printf("Ciência Comp.\n"); break; default:printf("Opção inválida\n"); } printf("Matrícula de %s OK!\n", nome); } else if (idade < 17 && nota < 60) { printf("Negada: idade e nota.\n"); } else if (idade < 17) { printf("Negada: menor de idade.\n"); } else { printf("Negada: nota insuficiente.\n"); } return 0; }

🟠 Java

import java.util.Scanner; public class Matricula { public static void main(String[] args){ Scanner sc = new Scanner(System.in); System.out.print("Nome: "); String nome = sc.next(); System.out.print("Idade: "); int idade = sc.nextInt(); System.out.print("Nota: "); double nota = sc.nextDouble(); if (idade >= 17 && nota >= 60) { System.out.print( "1-EngSW 2-Sist 3-CC: "); int c = sc.nextInt(); switch(c) { case 1: System.out.println( "Eng. Software"); break; case 2: System.out.println( "Sistemas Info."); break; case 3: System.out.println( "Ciência Comp."); break; default:System.out.println( "Inválido"); } System.out.println( "Matrícula de "+nome+" OK!"); } else if (idade<17 && nota<60) { System.out.println("Negada: ambos."); } else if (idade < 17) { System.out.println("Negada: idade."); } else { System.out.println("Negada: nota."); } } }
