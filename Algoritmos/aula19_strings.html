🔵 C — formas de declarar

// Forma 1: literal (automático '\0') char nome[20] = "Tiago"; ← ✅ recomendado // Forma 2: caractere por caractere char s[4] = {'A','B','C','\0'}; // Forma 3: vetor sem inicialização char entrada[50]; ← lido depois com scanf // Forma 4: ponteiro (literal constante) char *msg = "Olá, mundo!"; ← só leitura! // ERRADO: sem espaço para '\0' char x[3] = "ABC"; ← sem '\0'! ❌ // ERRADO: atribuição direta char s[10]; s = "Olá"; ← ERRO! Use strcpy!

🔵 C — leitura e escrita

// scanf lê UMA palavra (para no espaço) char nome[50]; scanf("%s", nome); ← sem &! nome já é endereço printf("Olá, %s!\n", nome); // fgets lê linha INTEIRA (com espaços) char frase[100]; fgets(frase, 100, stdin); // fgets inclui '\n' no final! // Remover '\n' do fgets: frase[strlen(frase)-1] = '\0'; // puts imprime string + '\n' puts(nome); ← equivale a printf("%s\n",nome)

⚠️ scanf vs. fgets

scanf("%s") para no espaço — bom para uma palavra. fgets() lê a linha toda — melhor para nomes completos como "João da Silva".

char s[] = "Logica"; // Modo 1: índice (mais claro) for(int i=0; s[i] != '\0'; i++) printf("%c ", s[i]); // L o g i c a // Modo 2: while com '\0' int i = 0; while(s[i] != '\0') { printf("%c=%d ", s[i], s[i]); ← char e ASCII i++; } // Contar vogais int vogais = 0; for(int i=0; s[i]; i++) { ← s[i]!=0 char c = s[i] | 32; ← minúsculo if(c=='a'||c=='e'||c=='i' ||c=='o'||c=='u') vogais++; }

💻 Exemplo 1 — Validador de Senha

Percorrer a string testando cada caractere com funções do ctype.h

📌 Problema Intermediário

Validar uma senha: mínimo 8 caracteres, ao menos 1 maiúscula, 1 minúscula, 1 dígito e 1 símbolo especial. Informar o que está faltando.

🔵 C — validador completo

#include <string.h> #include <ctype.h> void validarSenha(char *senha) { int mai=0, min=0, dig=0, sim=0; int len = strlen(senha); for(int i=0; senha[i]; i++) { char c = senha[i]; if(isupper(c)) mai++; else if(islower(c)) min++; else if(isdigit(c)) dig++; else sim++; ← símbolo } printf("=== Análise: [%s] ===\n", senha); printf("Tamanho : %d %s\n", len, len>=8?"✅":"❌ (mín 8)"); printf("Maiúsc. : %d %s\n", mai, mai>0?"✅":"❌"); printf("Minúsc. : %d %s\n", min, min>0?"✅":"❌"); printf("Dígitos : %d %s\n", dig, dig>0?"✅":"❌"); printf("Símbolos: %d %s\n", sim, sim>0?"✅":"❌"); if(len>=8&&mai&&min&&dig&&sim) printf("🔐 Senha FORTE!\n"); else printf("⚠️ Senha FRACA.\n"); }

💻 Exemplo 2 — Inverter String e Palíndromo

Dois ponteiros caminhando em direções opostas — padrão clássico

📌 Inverter uma string Básico

🔵 C — inversão com dois ponteiros

void inverter(char s[]) { int ini = 0; int fim = strlen(s) - 1; while(ini < fim) { // Troca s[ini] e s[fim] char tmp = s[ini]; s[ini] = s[fim]; s[fim] = tmp; ini++; fim--; } } // "Londrina" → "anirdnoL" // "abcde" → "edcba" // Visualização: // L o n d r i n a // ↑ ↑ troca L e a // ↑ ↑ troca o e n // ↑ ↑ troca n e i // ↑ ↑ troca d e r

📌 Verificar palíndromo Intermediário

🔵 C — palíndromo ignorando case

int ehPalindromo(char s[]) { int ini = 0; int fim = strlen(s) - 1; while(ini < fim) { // Compara ignorando maiúsculas if(tolower(s[ini]) != tolower(s[fim])) return 0; ← não é ini++; fim--; } return 1; ← é palíndromo } // Testes: // "arara" → 1 ✅ // "Radar" → 1 ✅ (case ignored) // "hello" → 0 ❌ // "ABCBA" → 1 ✅

✅ Padrão dois ponteiros (ini/fim)

O padrão ini avança da esquerda, fim recua da direita resolve eficientemente inversão, palíndromo e outros problemas de strings. Complexidade O(n) com espaço O(1) — sem criar string auxiliar.

💻 Exemplo 3 — Tokenizar e Processar CSV

strtok divide strings por separadores — fundamental para dados reais

📌 Problema Avançado

Processar uma linha CSV: "João,25,Londrina,Engenharia" — separar os campos e exibir formatado.

🔵 C — tokenizar com strtok

#include <string.h> char linha[] = "Joao,25,Londrina,CC"; char *campos[10]; int n = 0; // Primeira chamada: passa a string char *tok = strtok(linha, ","); // Chamadas seguintes: passa NULL while(tok != NULL) { campos[n++] = tok; tok = strtok(NULL, ","); } // Usar os campos: printf("Nome : %s\n", campos[0]); ← Joao printf("Idade : %s\n", campos[1]); ← 25 printf("Cidade: %s\n", campos[2]); ← Londrina printf("Curso : %s\n", campos[3]); ← CC // Converter campo para inteiro: int idade = atoi(campos[1]); ← "25"→25 printf("Maior de 18? %s\n", idade>=18?"Sim":"Não");

⚠️ Como strtok funciona

strtok modifica a string original — substitui os separadores por '\0'. Por isso trabalha com uma cópia!

Primeira chamada: passa a string. Próximas chamadas: passa NULL — strtok lembra onde parou internamente (variável estática).

🔵 C — funções de conversão

#include <stdlib.h> // String → número int x = atoi("42"); ← 42 float f = atof("3.14"); ← 3.14 long l = atol("99999"); ← 99999L // Número → String char buf[20]; sprintf(buf, "%d", 42); ← "42" sprintf(buf, "%.2f", 3.14);← "3.14" // Comparação case-insensitive (Linux) strcasecmp("ABC", "abc"); ← retorna 0

💻 Exemplo 4 — Cifra de César

Criptografia simples: deslocar cada letra no alfabeto — aritmética ASCII

📌 Problema Intermediário

Cifra de César: desloca cada letra N posições no alfabeto. A(+3)=D, Z(+3)=C (circular). Implemente codificar e decodificar.

🔵 C — cifra de César

void cifrar(char s[], int chave) { for(int i=0; s[i]; i++) { char c = s[i]; if(isupper(c)) s[i] = (char)((c - 'A' + chave) % 26) + 'A'; else if(islower(c)) s[i] = (char)((c - 'a' + chave) % 26) + 'a'; // Não-letras ficam inalteradas } } void decifrar(char s[], int chave) { cifrar(s, 26 - (chave % 26)); // Decifrar = cifrar com 26-chave } // Exemplo com chave=3: char msg[] = "ATACAR AO AMANHECER"; cifrar(msg, 3); printf("%s\n", msg); ← "DWDFDU DR DPDQKHFHU" decifrar(msg, 3); printf("%s\n", msg); ← "ATACAR AO AMANHECER"

🔢 A mágica da aritmética circular

Para mover a letra dentro do alfabeto circularmente:

(c - 'A' + chave) % 26 + 'A'

Passo 1: c - 'A' → posição 0..25
Passo 2: + chave → desloca
Passo 3: % 26 → circular (Z+1 = A)
Passo 4: + 'A' → volta para ASCII

✅ Trace: 'Y' com chave 3

'Y' - 'A' = 24
24 + 3 = 27
27 % 26 = 1
1 + 'A' = 'B' ✅ (Y → B, circular!)

🌐 Strings no Portugol e no Java

A mesma ideia em linguagens diferentes — abstrações de nível mais alto

🟢 Portugol — cadeia

var nome : cadeia frase : cadeia tamanho : inteiro inicio nome <- "Tiago" escreval("Olá, " + nome + "!") // Comprimento tamanho <- Comprimento(nome) escreval("Tamanho: ", tamanho) // Concatenar com + frase <- "Bem-vindo, " + nome // Converter para maiúsculo escreval(Maiusculo(nome)) // Parte da string (posições 1..3) escreval(Copia(nome, 1, 3)) ← "Tia" // Posição de substring escreval(Pos("ago", nome)) ← 3 fimalgoritmo

🟠 Java — classe String

String nome = "Tiago"; String frase; // Comprimento int len = nome.length(); ← 5 // Concatenar frase = "Olá, " + nome + "!"; // Maiúsculas / minúsculas nome.toUpperCase() ← "TIAGO" nome.toLowerCase() ← "tiago" // Comparar (equals, não ==!) nome.equals("Tiago") ← true nome.equalsIgnoreCase("tiago") ← true // Substring nome.substring(0, 3) ← "Tia" nome.charAt(0) ← 'T' nome.indexOf("ago") ← 2 nome.contains("ago") ← true nome.replace("i","y")← "Tayago"

✅ C vs. Portugol vs. Java

Em C: string = vetor de char, controle manual, funções em string.h.  |  Em Portugol: tipo cadeia, funções embutidas (Comprimento, Maiusculo, Copia), sem '\0' visível.  |  Em Java: classe String imutável, métodos encadeáveis, equals() obrigatório — nunca ==!

🚨 Erros Comuns com Strings

Os bugs mais frequentes — e como evitá-los

❌ Erro 1: Comparar com ==

// C — == compara ENDEREÇOS if(nome == "Tiago") ← SEMPRE false! // CORRETO: if(strcmp(nome, "Tiago") == 0) // Java — == compara referências if(str == "hello") ← inseguro! // CORRETO: if(str.equals("hello"))

❌ Erro 2: Buffer overflow (sem limite)

// ERRADO: usuário pode digitar 200 chars! char nome[10]; scanf("%s", nome); ← sem limite! // CORRETO: limitar com %Ns scanf("%9s", nome); ← máx 9 chars + '\0' // Ou usar fgets: fgets(nome, 10, stdin); ← seguro ✅

❌ Erro 3: Atribuir string com =

// ERRADO: = não copia string em C! char s[20]; s = "Londrina"; ← ERRO de compilação! // CORRETO: strcpy ou inicializar char s[20] = "Londrina"; ← na declaração ✅ strcpy(s, "Londrina"); ← depois ✅

❌ Erro 4: Esquecer o '\0' no tamanho

// "Londrina" = 8 chars char cidade[8] = "Londrina"; ← sem '\0'! // comportamento indefinido! // CORRETO: sempre N+1 char cidade[9] = "Londrina"; ← 8+1 ✅ // Ou deixe o compilador calcular: char cidade[] = "Londrina"; ← [9] auto ✅

✅ Checklist para strings em C

☑ Vetor tem tamanho N+1 para N chars?  |  ☑ Comparar com strcmp(), nunca ==?  |  ☑ Copiar com strcpy(), nunca =?  |  ☑ Leitura com limite (%9s ou fgets)?
